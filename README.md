# Alura Store LATAM - Análisis de Ventas y Recomendación Estratégica

Este proyecto presenta un análisis exploratorio de datos de ventas de la cadena **Alura Store** en América Latina. A través del uso de herramientas de análisis de datos y visualización geográfica, se estudian múltiples factores clave para tomar decisiones estratégicas sobre cuál tienda representa la mejor oportunidad comercial.

##  Análisis realizados

El notebook contiene análisis detallados por tienda en las siguientes áreas:

- **Facturación total** 
- **Productos más y menos vendidos**.
- **Categorías con mayor y menor rotación**.
- **Calificaciones promedio de los clientes**.
- **Costo promedio de envío por tienda**.
- **Visualización geográfica** interactiva con `plotly` y `folium`.

---

##  Conclusión del análisis

Tras considerar todos los factores, se concluye que la **Tienda 3** representa la mejor oportunidad para que el Sr. Juan, dueño de Alura Store, concentre sus esfuerzos de venta. Esta tienda destaca por:

- La **mayor facturación total**.
- Las **mejores calificaciones de clientes**.
- Una **buena diversidad y rotación de productos**.
- El **menor costo promedio de envío**.
